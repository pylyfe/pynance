Visualizing Volatility Using Matplotlib¶

In this project we are mainly going to use matplotlib to create some graphs out of time-series stock data. We will be analyzing both daily returns and the volatility of FANG (Facebook, Apple, Netflix, Google) portfolio, and then graphing the volatility using matplotlib.

To start with, let's make the necessary imports:

  • Pandas (for dataframes and data analysis)
  • Matplotlib (for graphing purposes)
  • Datetime (convert datetime in a format that can be processed by Pandas)

As with most financial applications using Python, these are the three most important libraries to have at your disposal.

import pandas as pd

First, we will specify the range of time we want to pull our data out from.

In [2]:
start_date = dt.datetime(2017,1,1)
end_date = dt.datetime(2018,1,1)

Next, we create a list of ticker symbols for our FANG assets, and then create an empty dataframe where we can store stock data we pulled out from pandas.

Note that we will only be needing the "Adjusted Close" prices from each stock to perform our analysis

In this project, we will be pulling out data from the Quandl API, during the time frame specified above. (1 year's worth)

In [3]:
assets = ['FB', 'AAPL', 'NFLX', 'GOOGL']

df = pd.DataFrame()

for stock in assets: 
    df[stock] = web.DataReader(stock, data_source = 'quandl', start = start_date, end = end_date)['AdjClose']

In order to prepare the dataframe for our analysis, we actually need the daily returns of each stock (expressed as a percentage change).

After we get the daily returns, we can use it to also get the daily volatilities of each asset using pandas.

In [5]:
# Get the daily returns (percentage value)
asset_returns_daily = df.pct_change()

# Get the daily volatilities
asset_daily_volatility = asset_returns_daily.std()

Note that the first row in daily returns is NaN. This is because in order to compute daily returns you need at least 2 periods (in this case days) of data, but there is no row above the first row for us to calculate it.

In this case since we are only graphing them using matplotlib it will not be a problem. However in the future please note that for other analysis we will have to remove the first row.

Now we will use matplotlib to plot the daily returns.
